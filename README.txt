Instructions:
1. Copy your assets/ folder (images & videos) into public/assets/
2. Place master.env in project root (same folder as server.js). Example master.env contents:
   OPENAI_API_KEY=sk_xxx
   GOOGLE_SHEET_CSV=https://docs.google.com/.../pub?output=csv
   DEFAULT_CSV=https://docs.google.com/.../pub?output=csv
3. Optionally copy tour_template.csv to project root.
4. Install & run:
   npm install
   npm start
5. Open http://localhost:3000
